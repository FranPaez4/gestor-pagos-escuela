Gestor de Pagos Escolar
Proyecto full-stack desarrollado como práctica para la gestión administrativa y control de pagos de un centro educativo. Esta aplicación permite gestionar alumnos, registrar mensualidades y controlar la morosidad de forma sencilla e intuitiva.

Tecnologías Utilizadas
Este proyecto está dividido en dos partes principales, utilizando un stack tecnológico moderno:

Frontend:

React

Vite

Tailwind CSS

JavaScript

Backend:

Java

Spring Boot

PostgreSQL

Maven

 Características Principales
Gestión de Usuarios: Creación, lectura, actualización y borrado (CRUD) de alumnos.

Control de Pagos: Registro de cuotas mensuales y generación de recibos.

Interfaz Responsiva: Diseño moderno y adaptable a cualquier dispositivo gracias a Tailwind CSS.

API REST: Comunicación fluida y segura entre el cliente y el servidor.

 Instalación y Ejecución en Local
Si deseas clonar y probar este proyecto en tu entorno local, sigue estos pasos:

Prerrequisitos
Node.js y npm instalados.

Java Development Kit (JDK) 17 o superior.

Servidor PostgreSQL instalado y en ejecución.

Configuración del Backend
Entra en la carpeta Backend.

Abre el proyecto con tu IDE favorito.

Configura las credenciales de tu base de datos PostgreSQL en el archivo application.properties o application.yml.

Ejecuta la aplicación principal mediante Maven. El servidor iniciará en el puerto 8080.

Configuración del Frontend
Abre una terminal y navega hasta la carpeta Frontend.

Instala las dependencias ejecutando el comando: npm install

Inicia el servidor de desarrollo ejecutando: npm run dev

Abre la URL proporcionada (normalmente http://localhost:5173) en tu navegador.
